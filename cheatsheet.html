HTML Tables

Table Tags

HTML Tag Name Description
<table> Table Wraps all of the tags and data for HTML tables
<thead> Table Head Contains all of the table header elements and data
<tbody> Table Body Contains all of the table body elements and data
<tfoot> Table Footer Contains all of the table footer elements and data
<th> Table Heading Describes the data in a data column. Goes in the Table Head
<tr> Table Row Contains rows of table data
<td> Table Data Actual data that appears on the table; goes in the Table Row container

Table Attributes

Attribute Name Description
colspan Column Span Determines how many columns a <td> element takes up
rowspan Row Span Determines how many rows a <td> element takes up
scope Scope Defines the scope of <th> elements and how it relates to the associated <td> elements. Accepts row, col, and others
